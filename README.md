Task 1 – Spring Boot Application
This project was created as part of Task 1 for the "Spring Framework Apps" course at Akademia Finansów i Biznesu Vistula.

The application was built using **Spring Boot** and **Maven** and demonstrates basic handling of HTTP requests as well as the MVC pattern.

Technologies
- Java 17
- Spring Boot
- Spring Web
- Thymeleaf
- Maven

Returns a simple text response:

"Hello Vistula, my first Spring controller!"

Returns an HTML page rendered using **Thymeleaf** with a greeting message.

How to Run
1. Run the `Task1Application` class
2. Open `http://localhost:8080` or `http://localhost:8080/greeting` in a browser

Task 1 completed successfully.
